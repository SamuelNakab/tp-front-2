Vinilos — Catalogo musical personal
Aplicacion web para registrar y gestionar tu coleccion de albumes. Cada usuario tiene su propia cuenta y su propia coleccion separada.
Que hace: desde el dashboard ves todos tus albumes en grilla con portada, genero, año y rating. Podes agregar albumes nuevos con titulo, artista, año, genero, rating de 1 a 5 estrellas, resena y foto de portada. Tambien editarlos o borrarlos. Al crear uno nuevo se dispara una animacion de notas musicales como confirmacion visual.
El acceso es por registro con email y contraseña via Supabase, que tambien maneja la base de datos y el storage de las imagenes.
